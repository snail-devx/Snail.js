# 1、基础简介

1. 作为 `snail` 系列的核心包，提供最基础的功能组件，和框架无关，跨web、node等js运行时使用
2. 提供数据类型扩展、数据类型校验、事件管理、网络请求、资源版本等等

# 2、功能目录

| 目录名称 | 用途说明   | 备注说明                                                                                                 |
| -------- | ---------- | -------------------------------------------------------------------------------------------------------- |
| base     | 基础核心   | -  对js自身数据类型等的扩展（array、error、number、、、）<br />-  提供格式化、校正、校验、类型判断等功能 |
| common   | 基础通用   | 一些通用功能组件，如事件、自定义数据类型、、、                                                           |
| thread   | 多线程相关 | 如并发锁、线程相关操作组件                                                                               |
| web      | 网络编程   | 网络请求，网络服务器管理、url格式化、资源版本等                                                          |
|          |            |                                                                                                          |
